O que é React ?

R.:  é uma biblioteca JavaScript de código aberto, criada pelo Facebook em 2013, focada na construção de interfaces de usuário (UIs) interativas e reativas para aplicações web e móveis

O que é um componente React ?

R.: é uma função JavaScript (ou classe) independente e reutilizável que retorna elementos HTML, agindo como blocos de construção para interfaces de usuário

O que são arquivos .tsx ?

R.: são arquivos JavaScript que utilizam uma extensão de sintaxe chamada JSX (JavaScript XML), permitindo escrever estruturas semelhantes a HTML diretamente no código React. Eles facilitam a criação de componentes de interface, unindo marcação e lógica em um único arquivo, sendo posteriormente convertidos em JavaScript puro para o navegador.
